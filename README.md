# Sample Gym - Premium Multi-Page Website

Static production-ready front-end scaffold for a premium black-and-gold gym website.

## Run

Open `index.html` directly in browser, or serve locally:

```powershell
Set-Location "d:\gym sample"
python -m http.server 5500
```

Then open http://localhost:5500

## Included Pages

- Home
- About
- Trainers
- Services
- Membership
- Register (multi-step)
- Payment
- Success
- Gallery
- Schedule
- Contact
- Blog
- Testimonials
- Location
- Admin (UI placeholder)

## Implemented UX Systems

- Black/gold design token system
- Fixed navbar with mobile slide menu
- Scroll reveal animations
- Counter animation
- Testimonials auto-rotate
- Membership monthly/annual toggle
- Registration multi-step form flow
- Cookie consent storage in localStorage
- Floating WhatsApp + scroll-to-top
- Toast notifications
- Mobile sticky CTA bar

## Production Integration Hooks

- Razorpay button currently simulated in `app.js`
- Admin login currently UI-only placeholder
- Contact and registration flows currently front-end only
